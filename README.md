# Paravane Labs open-source defaults

This repository contains the public organization profile and default community health files for Paravane Labs repositories.

The files here are used as shared defaults when an individual repository does not provide its own project-specific version.

## Contents

- `profile/README.md`: GitHub organization profile shown on the Paravane Labs profile page.
- `CODE_OF_CONDUCT.md`: baseline conduct expectations for public collaboration.
- `CONTRIBUTING.md`: general contribution guidance for public repositories.
- `SECURITY.md`: vulnerability reporting guidance.
- `LICENSE.md`: license for this repository's documentation and templates.

Individual repositories may include more specific contribution, security, support, or licensing terms. When they do, the repository-specific files take precedence.
